macOS에서 Safari 브라우저 자동화 기능을 사용하기 위한 설정 안내
=========================================================

이 프로그램을 macOS에서 Safari 브라우저와 함께 사용하려면, 최초 1회에 한해 Safari의 '원격 자동화' 옵션을 활성화해야 합니다.

아래의 절차를 따라 진행해주세요.

1.  **터미널(Terminal) 실행**
    - Spotlight 검색 (Cmd + Space)에서 'Terminal'을 검색하여 실행하거나,
    - '응용 프로그램' > '유틸리티' 폴더에서 '터미널'을 찾아 실행합니다.

2.  **활성화 명령어 입력**
    - 열려있는 터미널 창에 아래의 명령어를 그대로 입력하고 Enter 키를 누릅니다.

      ```bash
      safaridriver --enable
      ```

3.  **관리자 암호 입력**
    - 명령어 실행 시 macOS 사용자 계정의 암호를 요구할 수 있습니다.
    - 암호를 입력할 때 화면에 글자가 표시되지 않는 것은 정상이니, 그대로 암호를 입력하고 Enter 키를 누릅니다.

4.  **설정 완료**
    - 위 과정이 성공적으로 완료되면 이제 이 프로그램을 통해 Safari 브라우저를 제어할 수 있습니다.
    - 이 설정은 한 번만 해주시면 됩니다.

이제 Selenium UI Test Tool을 실행하여 Safari 로그인 및 스크린샷 기능을 사용할 수 있습니다.
